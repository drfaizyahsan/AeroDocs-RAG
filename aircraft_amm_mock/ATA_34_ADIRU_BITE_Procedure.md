# AIRCRAFT MAINTENANCE MANUAL
## ATA 34-21-00
### ADIRU BUILT-IN TEST EQUIPMENT (BITE)

System:
Air Data Inertial Reference Unit

### Entry Conditions

- Aircraft electrical power ON
- IRS mode selector: NAV
- Parking brake SET

### Procedure

1. Access MCDU maintenance menu.
2. Select:
   CFDS > AVIONICS > ADIRU

3. Execute:
   GROUND SCANNING TEST

4. Record fault codes.

### Typical Fault Codes

| Code | Description |
|------|-------------|
| ADR101 | Air Data Sensor Failure |
| IRU204 | Gyro Alignment Error |
| IRU331 | Accelerometer Fault |

### Acceptance Criteria

PASS:
- No active faults

FAIL:
- Active fault remains after reset cycle
