# AIRCRAFT MAINTENANCE MANUAL
## ATA 49-21-13
### APU OIL FILTER REPLACEMENT

APU Model:
GTCP131-9A

### References

- AMM 49-00-00
- CMM 49-21-13

### Removal

WARNING:
APU MUST BE SHUT DOWN FOR A MINIMUM OF 60 MINUTES.

1. Open APU access door.
2. Place drip tray.
3. Remove oil filter assembly.
4. Collect residual oil.

### Inspection

Inspect filter element for:
- Ferrous particles
- Carbon deposits
- Metallic contamination

### Installation

Part Number:
- Filter: 23065874-001

Torque:
- Filter housing: 32 ±2 Nm

### Operational Check

1. Start APU.
2. Run for 5 minutes.
3. Verify:
   - Oil pressure: 55-70 psi
   - No leaks
   - Normal EGT indication
