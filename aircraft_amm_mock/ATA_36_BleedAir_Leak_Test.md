# AIRCRAFT MAINTENANCE MANUAL
## ATA 36-11-00
### BLEED AIR SYSTEM LEAK TEST

### Equipment

- Pneumatic pressure source
- Ultrasonic leak detector
- Pressure gauge calibrated to ±1%

### Test Conditions

| Parameter | Value |
|-----------|-------|
| Test Pressure | 45 psi |
| Ambient Temperature | 15°C to 35°C |
| Test Duration | 10 minutes |

### Procedure

1. Isolate engine bleed valves.
2. Connect external pneumatic source.
3. Pressurize system to 45 psi.
4. Monitor pressure decay.

### Acceptance Limits

Maximum pressure loss:
- 2 psi over 10 minutes

If pressure loss exceeds limits:
- Perform ultrasonic inspection.
- Repair defective joints.
- Repeat test.
