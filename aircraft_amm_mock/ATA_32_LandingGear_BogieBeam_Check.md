# AIRCRAFT MAINTENANCE MANUAL
## ATA 32-11-15
### MAIN LANDING GEAR BOGIE BEAM INSPECTION

Inspection Method:
- Eddy Current Inspection
- Visual Inspection

### Areas of Interest

- Axle transition radius
- Torque link attachment points
- Bogie pivot housing
- Brake rod attachment lugs

### Crack Acceptance Criteria

| Crack Length | Action |
|--------------|--------|
| <2.0 mm | Monitor |
| 2.0-5.0 mm | Engineering Review |
| >5.0 mm | Replace Component |

### NDT Requirements

Equipment:
- Eddy Current Probe: 500 kHz
- Calibration Block: EC-CAL-32

Acceptance:
No linear indications exceeding threshold level 3.
